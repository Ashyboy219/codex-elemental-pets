Official desktop path

Install through Codex.

Open the pets folder from Settings so Codex chooses the right location for your machine. Read the official Pets docs ↗

  1. 1

    Download the drag-and-drop pet pack.

  2. 2

    In Codex, open Settings → Pets → Custom pets → Open folder.

  3. 3

    Copy the frostbyte and/or bolt folder from the pack into that folder.

  4. 4

    Return to Pets, choose Refresh, then select your new companion.
